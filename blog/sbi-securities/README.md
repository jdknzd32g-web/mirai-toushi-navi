# SBI証券 記事制作メモ

このフォルダは、未来投資navi内で展開するSBI証券関連の記事群を管理する場所です。

## URL方針

SBI証券の記事は年別フォルダではなく、以下の独立カテゴリ配下にまとめます。

```text
/blog/sbi-securities/
```

年別フォルダに入れると、記事内容が古く見えやすくなります。
SBI証券、NISA口座、iDeCo、住信SBIネット銀行連携は継続的に更新するテーマなので、カテゴリ型のURLで運用します。

## 位置づけ

未来投資naviの読者は、主に50代・60代の新NISA初心者です。
SBI証券の記事は「証券会社の網羅比較」ではなく、FPりょう自身が実際にSBI証券を使い、お客様にも操作方法を案内している経験をもとに作ります。

## 基本スタンス

- 他社をすべて使い込んだ比較記事にはしない
- 「僕自身が使っていて、自信を持って案内できる証券会社」として紹介する
- 楽天証券など他社を否定しない
- SBI証券の画面は最初やや分かりづらい、という本音も書く
- ただし、NISA、iDeCo、住信SBIネット銀行、クレカ積立、将来的なCFDなどを一体で使える点を強みにする
- CFDは初心者向けの主訴求にしない

## 成約導線

読者の流れは以下を想定します。

1. 新NISAを始めたいが、証券会社選びで迷っている
2. SBI証券は難しそう、使いにくそうという不安がある
3. FPりょうが実際に使っている理由を読む
4. 口座開設手順記事で不安を解消する
5. 公式サイトへ進む

## アフィリエイトリンク

アクセストレードのSBI証券リンク:

```text
https://h.accesstrade.net/sp/cc?rk=0100poxk00nng7
```

リンク文言は、過度に煽らず以下を基本にします。

- SBI証券の公式サイトを確認する
- SBI証券でNISA口座を確認する
- SBI証券の口座開設ページを見る

## 注意

金融商品や個別銘柄の購入を勧誘する記事にはしません。
証券口座、NISA口座、iDeCo、クレカ積立などの使い方と選び方を中心にします。

## ヘッダー画像方針

未来投資naviのブログヘッダーは、写真風の16:9画像で統一します。
ただし、単に「50代・60代が投資資料を見ている写真」に寄せると過去記事と似すぎるため、記事内容ごとに場面、小物、構図を変えます。

共通ルール:

- 写真風、自然光、落ち着いた生活感のある雰囲気にする
- 50代・60代の読者が自分ごと化しやすい人物や手元を入れる
- 文字、ロゴ、ブランド名、実在企業の画面UIは入れない
- SBI証券ロゴや実在サービスの画面をそのまま出さない
- 記事本文の主題が一目で伝わる小物を入れる
- 過去記事と同じ「夫婦が資料とPCを見ているだけ」の構図を繰り返さない

記事別の方向性:

- 総合レビュー記事: ネット証券の初期設定、口座開設準備、PCとスマホ、本人確認書類、メモ帳
- 口座開設手順記事: スマホで本人確認、マイナンバーカード風のカード、チェックリスト、PC入力画面
- 新NISA記事: NISA積立設定、家計ノート、毎月の積立額を考える手元
- iDeCo記事: 老後資金の計画表、年金書類風の紙、夫婦または個人が将来設計を確認する場面
- 住信SBIネット銀行連携記事: 銀行アプリ風の画面、入出金メモ、家計口座と投資口座の整理
- デメリット記事: 画面の情報量に少し迷っている表情、メモを取りながら設定を確認する場面

プロンプトの基本形:

```text
写真風の16:9ブログヘッダー画像。未来投資naviの既存記事と同じ落ち着いた金融・家計相談のトーン。
ただし、過去記事と同じ「夫婦が投資資料とPCを見ているだけ」の構図は避ける。
この記事の主題は「{記事テーマ}」なので、{記事テーマに合う小物・動作・場面}を中心に構成する。
50代・60代の日本人読者が自分ごと化できる自然な生活感。温かい自然光、木製の机、清潔な室内。
PCやスマホには汎用的な金融ダッシュボード風の画面を表示するが、実在企業名、ロゴ、SBI証券の画面、読める文字は入れない。
文字入れ、ロゴ、ブランド名、過度な笑顔、イラスト調、YouTubeサムネ風の派手な加工は避ける。
```
